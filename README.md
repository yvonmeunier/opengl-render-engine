# What is this project ?
This is a simple render engine made by following learnopengl.com

# Project Setup
You must provide the following dlls if you want to use another compiler than g++ from mingw64 on Windows: 
- libglfw3dll.a in the /lib folder
- glfw3.dll in the /bin folder

For MacOS you must provide :
- libglfw.X.dylib in both /lib and /build/lib
- libglfw3dll.a in the /lib folder


Both folders are at the root of the project.
