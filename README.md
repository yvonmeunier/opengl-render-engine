# What is this project ?
This is a simple render engine made by following learnopengl.com

# Project Setup
You must provide the following dlls : 
- libglfw3dll.a in the /lib folder
- glfw3.dll in the /bin folder

All compiled using g++.exe from mingw64.
Both folders are at the root of the project.

If you want to use another compiler or use another OS, you must edit the build task accordingly and provide the proper dlls otherwise it won't work.
